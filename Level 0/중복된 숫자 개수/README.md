<b> <h2> 문제 설명 </h2></b>
정수가 담긴 배열 array와 정수 n이 매개변수로 주어질 때, array에 n이 몇 개 있는 지를 return 하도록 solution 함수를 완성해보세요.
<br>
<b> <h2> 제한사항 </h2></b>
<li> 1 ≤ array의 길이 ≤ 100 </li>
<li> 0 ≤ array의 원소 ≤ 1,000 </li>
<li> 0 ≤ n ≤ 1,000 </li>
<br>
<b> <h2> 입출력 예 </h2></b>
<table>
  <tr>
    <td> array </td>
    <td> n </td>
    <td> result </td>
  </tr>
  <tr>
    <td> [1, 1, 2, 3, 4, 5] </td>
    <td> 1 </td>
    <td> 2 </td>
  </tr>
  <tr>
    <td> [0, 2, 3, 4] </td>
    <td> 1 </td>
    <td> 0 </td>
  </tr>
</table>
<br>
<b><h2> 입출력 예 설명 </h2></b>
<h3> 입출력 예 #1 </h3>
<li> [1, 1, 2, 3, 4, 5] 에는 1이 2개 있습니다. </li>
<h3> 입출력 예 #2 </h3>
<li> [0, 2, 3, 4] 에는 1이 0개 있습니다. </li>
